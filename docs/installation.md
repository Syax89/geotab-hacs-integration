## Method 1: HACS (Recommended)
HACS (Home Assistant Community Store) is the best way to keep the integration updated.

1.  **Open HACS** in your Home Assistant instance.
2.  Navigate to **Integrations**.
3.  Click the three dots in the top right corner and select **Custom repositories**.
4.  Paste the repository URL: `https://github.com/Syax89/geotab-hacs-integration`
5.  Select **Integration** as the category and click **Add**.
6.  Search for "Geotab" and click **Install**.
7.  **Restart Home Assistant**.

## Method 2: Manual Installation
1.  Download the latest [release](https://github.com/Syax89/geotab-hacs-integration/releases).
2.  Extract the archive and locate the `custom_components/geotab` folder.
3.  Copy this folder into your Home Assistant `custom_components` directory.
4.  Restart Home Assistant.
